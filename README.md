# GoT Fan page

This repo has an example of a GoT fan page created for Basics of Web Development course, spring 2025, week 02, workshop task.

Simple command line instructions to get the code working on your laptop and start editing:

```bash
git clone git@github.com:Zaiban/got-fanpage.git
cd got-fanpage/frontend
npm install
npm run start
```
