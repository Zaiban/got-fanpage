# GoT Fan page

This repo has an example of a GoT fan page created for Basics of Web Development course, spring 2025, week 02, workshop task.

Simple command line instructions to get the code working on your laptop and start editing:

```bash
git clone git@github.com:Zaiban/got-fanpage.git
cd got-fanpage/frontend
npm install
npm run start
```

The actual code for the page can be found in: `got-fanpage/frontend/src/App.js`.

Note that this site is created with the "React.js" library, which is a more advanced library for creating
web applications. React is not in the scope of the Course. To finish the workshop task yourself, you should
learn & follow Moodle materials written by Louis Botha, and use technologies explaines during weeks 47-49 and week 2.
